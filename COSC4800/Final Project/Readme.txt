Final Project
"Restaurant Advisor"

Project consists of 2 parts
a) NodeJS Backend Server
b) Android Frontend App

A takes HTTP queries and returns JSON SQL tables.
B makes HTTP requests to server and displays data in a friendly form.

Where to find code:
a) Android/Java code is available in RestaurantAdvisor/app/src/main/
	Java code is available under java/com/mail929/android/restaurantadvisor
	XML resources are available under res/
b) (Significant) Server JavaScript code is available in Node/api/
	request.js handles SQL requests
